Players

PlayerType: 1=Human, 2=Computer, 3=Barbarians
Build/Unit Type: 0=Greek, 1=Egypt, 2=Indian, 3=Messo, 4=Chineese, 5=Celt, 6=Barbarian
Player=0,"Homer",4,1,4


Player Has Lost - 1 indicates yes, 0 no. Each corresponds to player id. 
In this example, players 1 and 2 have lost. Players 0, 3, 4 are still in.
lost=0,1,1,0,0

Each value represents one of the knowledge score areas

Buildings - Max
There are 10 buildings. Each 2 number set represents the level and culture of building. (3 = level 3) The culture is tied to the player ID. You can only select and ID for which there is a player. Selecting a non-player number will result in an error.
